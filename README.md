# Random Background Color Generator

This project is a simple and interactive **Random Background Color Generator** built using **HTML**, **CSS**, and **JavaScript**.  
With each button click, the webpage background changes to a unique random color generated through a JavaScript function.

---

## üöÄ Features
- Generates a completely random color on each click  
- Applies the color dynamically to the page background  
- Stylish gradient button and decorative heading  
- Clean, attractive UI with hover effects  
- Lightweight and responsive

---


## üß† How It Works
- The `RandomColor()` function creates a hex color using random values.
- JavaScript applies that color to:
  - `document.body.style.backgroundColor`
- A Click Event Listener triggers the color change each time the user presses the button.

## How to Run the Project

1. Download or clone the repository
2. Open the folder
3. Simply open **index.html** in any browser
